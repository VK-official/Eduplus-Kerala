Eduplus Kerala is a fast, modern, and 100% open-source educational platform designed specifically for the students and teachers of Kerala following the SCERT syllabus. 

Built with transparency and community in mind, this platform serves as a secure gateway to official study materials and a collaborative space for verified educators to share resources.

## Key Features

* **KITE Samagra Gateway:** A clean, intuitive filter system that directly connects students to the official SCERT Kerala textbooks (Classes 5-10, English & Malayalam Mediums) ensuring they always have the latest government-approved materials.
* **The Teacher Vault:** A secure, authenticated portal where verified educators can upload, manage, and share high-quality study notes, previous year question papers, and resource links.
* **Real-Time Feedback Engine:** A globally accessible feedback widget that allows students and teachers to report issues or suggest features directly to the platform administrators.
* **Blazing Fast UI:** Built on the Next.js App Router for instant page loads, featuring a modern, dark-themed UI with native Malayalam typography support.
* **Open Source & Privacy First:** No trackers, no hidden fees, and no paywalls. The entire codebase is public for complete transparency.

## Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Database & Auth:** [Supabase](https://supabase.com/)
* **Hosting:** [Vercel](https://vercel.com/)
* **Typography:** Noto Sans Malayalam (Google Fonts)
